FB Page Manager

Privacy Policy

Last updated: March 22, 2026
FB Page Manager is a personal desktop application that helps individuals manage their own Facebook Pages. This policy explains what data we access, how we use it, and how we protect it.

1. What This App Does

FB Page Manager is a desktop tool that allows users to:

  • Connect their own Facebook Page using a Page Access Token
  • Publish text posts and video content to their own Facebook Page
  • Schedule posts and videos to be published at specific times
  • Manage a local queue of video files from their own computer
This app is a single-user tool. Each user connects only their own Facebook Page. No data is shared between users.

2. Data We Access

The app requests the following Facebook permissions:

  • pages_show_list — to identify the pages the user manages
  • pages_manage_posts — to publish posts and videos to the user's own page

We do not access, read, or store any data about the user's friends, followers, personal profile, messages, or any pages other than the one they explicitly connect.

3. How We Store Data

All data is stored locally on the user's device only. This includes:

  • Facebook Page Access Token
  • Post history and scheduled posts
  • App settings (description, hashtags, schedule times)
  • Video queue metadata (file names and paths)
No data is sent to any external server. No database, no cloud storage, no analytics. Everything stays on the user's own computer.

4. Video Content

When a user schedules a video for upload, the app reads the video file directly from the user's local computer and uploads it to Facebook on the user's behalf using the Facebook Graph API. The video is not stored or processed by any third party other than Facebook.

5. Third-Party Services

This app uses the Facebook Graph API to publish content. All interactions with Facebook are governed by Facebook's Privacy Policy and Facebook's Platform Terms.

No other third-party services are used.

6. Data Deletion

Users can delete all locally stored data at any time by:

  • Clicking "Disconnect" inside the app — this clears the stored token and all settings
  • Uninstalling the application — this removes all stored data from the device

To request deletion of any data that Facebook holds as a result of using this app, users should use Facebook's data deletion tools.

7. Children's Privacy

This application is not directed at children under the age of 13. We do not knowingly collect any personal information from children.

8. Changes to This Policy

We may update this privacy policy from time to time. Any changes will be reflected on this page with an updated date.
